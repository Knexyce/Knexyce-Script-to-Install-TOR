# Knexyce Script to Install TOR

What is TOR/TOR Browser?

TOR Browser is a browser that allows you to anonymously browse the Internet with more security. It also allows you to access sites and information that is normally hidden from the rest of the Internet. TOR Browser also has multiple layers of encryption to protect privacy and a feature called NoScript to protect you from being hacked. Keep in mind that this browser is in the Deep Web and is often called the Dark Web. Be cautious when using it and remember to have as much security as possible before connecting to your Wi-Fi/Ethernet/etc. if you intend to use this browser.

## Requirements
1. A Linux-based operating system. (Debian/Ubuntu/ChromeOS/etc. recommended)
2. Bash shell.
3. Enabled Linux Terminal.

## Features
1. Updates your package list and upgrades existing packages.
2. Installs necessary dependencies for TOR Browser.
3. Fetches the latest version of TOR Browser's archive format and extracts it.
4. Launches TOR Browser automatically after installation.
5. Provides information at the end of the process.

To use the code:

1. Download "install_tor.sh" from this repository.
2. Move the "install_tor.sh" file to the "Linux files" folder or wherever your terminal has access to.
3. Open the terminal and type these 2 commands:

- chmod +x install_tor.sh
- ./install_tor.sh

4. To launch TOR again: Repeat Step-3.
Note: DO NOT MODIFY/DELETE ANY FOLDER CONTAINING THE KEYWORD: "tor-browser" It may damage TOR Browser and/or your system. This also applies to anything in the folder containing "tor-browser" in the folder's name.

## Additional Information

This repository provides a simple script to remotely install and launch the official TOR Browser. For more detailed information about the browser, visit [TOR Project’s official website](https://www.torproject.org/).
