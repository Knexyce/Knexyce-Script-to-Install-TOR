# Knexyce-Script-to-Install-TOR
Installs TOR remotely.

To use the code:

1. If not already done so, enable Linux Dev. Tools/Environment.
2. Download "install_tor.sh" from this repository.
3. Move the "install_tor.sh" file to the "Linux files" folder.
4. Open the terminal and type these 2 commands:

# chmod +x install_tor.sh
# ./install_tor.sh

5. To launch TOR again: Repeat Step-4.
Note: DO NOT TOUCH/DELETE THE "tor-browser" FOLDER!

This repository is designed to provide a script that can remotely install and launch TOR Project's browser, go to their official site for more specifc details.
